Script Formats Supported

Zedata Block

Our modern, clean format designed specifically for Zedmonkey with enhanced security features and improved readability.

[script]
name = "Dark Mode Everywhere"
namespace = "zedmonkey"
version = "1.2.0"
description = "Applies dark mode to any website"
author = "Zedmonkey Team"
match = "*://*/*"
grant = "none"
