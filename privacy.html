Privacy Policy March 30, 2025

This privacy policy outlines how the Zedmonkey extension handles your data.

Data Collection

The Zedmonkey extension does not collect personal data from users. It operates locally within your browser and does not transmit data to external servers.

Permissions

The extension requires certain permissions to function effectively:

  • Storage: Used to save scripts and settings locally within your browser.
  • Scripting: Allows the extension to execute scripts on web pages as per user configuration.
  • ContextMenus: Enables the extension to add options to the browser's context menu for ease of use.
  • Host Permissions: The extension can access all URLs to apply scripts as configured by the user.

Local Storage

Zedmonkey stores the following information locally in your browser:

  • User scripts and their configurations
  • Extension settings and preferences
  • Script execution history (if enabled)

This data is stored using the browser's built-in storage APIs and is not accessible to other extensions or websites.

Cloud Synchronization

If you choose to use the cloud synchronization feature, your scripts and settings will be stored in your personal cloud storage account (Dropbox, Google Drive, etc.). Zedmonkey does not have access to your cloud storage credentials and uses OAuth for secure authentication.

Third-Party Scripts

Scripts installed through Zedmonkey may collect data according to their own privacy policies. We recommend reviewing the code of any script before installation to understand its data practices.

Changes to Privacy Policy

We may update this privacy policy from time to time. We will notify users of any significant changes by updating the date at the top of this policy.

Contact

If you have any questions about this privacy policy, please contact us through our GitHub repository.
